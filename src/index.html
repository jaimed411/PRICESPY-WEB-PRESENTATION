PriceSpy

La plataforma que combina analítica de datos e inteligencia artificial para proporcionar información valiosa sobre el mercado de viviendas en España y asi ayudar a la toma decisiones para determinar el valor de una vivienda

Provincias de España

Este es un modelo de Machine Learning alimentado con 85.000 datos que tiene como objetivo claro predecir el precio de la vivienda en las diversas provincias de España. ¿Cómo lo hacemos posible?

El proceso comienza con web scraping. Se recopila alrededor de 10800 anuncios de propiedades en cada provincia desde una web de viviendas. Cada provincia tiene su propio modelo único, adaptado a sus características de m2 y habitaciones.


Madrid y Málaga

Este es un modelo de ML más preciso con 40000 datos donde se predice el precio de la vivienda en la Comunidad de Madrid.

Después del scrapping y analizar todos los datos, hemos sacado un modelo donde es posible rellenar hasta 14 características de la vivienda para poder obtener un resultado más preciso.

Fraude de Anuncios de viviendas

Este es un modelo NLP con 2000 descripciones (1000 reales y 1000 fraudulentas generadas por inteligencia artificial).

Mediante la obtención de las descripciones de las viviendas obtenidas, y añadiendo textos falsos, hemos obetenido un modelo para identificar de manera precisa las características lingüísticas que indicaban posibles intentos de estafa en las descripciones de las viviendas.
